Slider

  • Require simpleslider.js.
  • Dont add col-md class next to the slider class. Slider class has width 100%, so it wont work. Wrap the slider in col-md class div instead.
  • Slide by default has background-cover class.
  • It is advise to use id to initialize the plugin, so each slider will be unique.
  • Alternate Indicator - extend this class ".slider-indicator.alt" with slideposition id.
  • Multiple slider - change the slideTrackerID option when intialize the plugin, else you will have repeated slideposition bullet.

Margin Mixin

Enable 1 line code for different direction margin. @include margin($t:1em,$r:1em,$b:1em,$l:1em)
The direction is optional, just put those you need. example: margin($t:1em,$b:3em)

Top 1em, Btm 3em Top 3em, Btm 1em, Left 2em Top 3em, Right 3em

Size Mixin

Shorthand for height and width. @include size($h,$w) If width not defined, it will auto inherit height.

Height and width both 20em
Height 10em,width 20em
Height 5em,width 20em

Position Mixin

Shorthand for position. First argurement is $p (position: absolute, relative, fixed), follow with $t, $r, $b, $l(top, left, bottom, right). Fixed will also work. I just dont want to messup the cheatsheet. @include position($p, $t, $r, $b, $l)

Position absolute, bottom 1em
Position relative, left 20em
Position absolute, right 1em, top 0
